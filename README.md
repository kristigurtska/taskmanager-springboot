# TaskManager 

Full stack task management app built with Java Spring Boot and SQL Server.

## Tech Stack
- Java 25 + Spring Boot 4
- Spring Data JPA + SQL Server
- HTML, CSS, JavaScript
- 
## Setup
1. Clone the repo
2. Create `taskmanager` database in SQL Server
3. Copy `application.properties.example` → `application.properties`
4. Fill in your credentials
5. Run `TaskManagerApplication.java`
6. Open `http://localhost:8080`
